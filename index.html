環境構築手順

前提

  1. WindowsにPythonをインストールしてVSCodeで開発する
  2. 開発したコードをGitHub上のリポジトリにpush
  3. pushされたコードを対象サーバーに自動的にpull

1.Windows上

  • Git for Windowsインストール
    →上記ページのDownload
  • VSCodeインストール
    →上記ページのWindows
  • Anacondaインストール
    →上記ページの右下のDownload
  • Anaconda Navigatorを起動してVSCodeをLaunch
  • VSCodeを起動して以下の拡張機能を追加
    • Japanese
    • Python
    • Python Extensions
    • Bracket Pair Colorizer 2
    • indent-rainbow
    • EvilInspector
  • VSCodeのターミナルを起動して ssh-keygen を実行
    →作成された ~/.ssh/id_rsa.pub を後ほどGitHubに登録する
  • 「2.GitHub上」の設定終了後、VSCodeのターミナル上で ssh -T git@github.com を実行
  • GitHub上のリポジトリをクローン

2.GitHub上

  • GitHubアカウントを作成してサインイン
  • 自分のSettings－SSH and GPG keysに上記公開鍵を登録
  • リポジトリ作成

3.EC2上

  • インスタンス作成
  • nginxインストール
